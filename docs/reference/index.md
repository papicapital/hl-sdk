# API Reference

This section provides detailed documentation for all classes, functions, and types in the Hyperliquid SDK.

## Core Classes

| Class | Description |
|-------|-------------|
| [Api](api.md) | Main entry point for the SDK. Provides access to all other components and handles initialization. |
| [Account](account.md) | Authentication credentials and account management for API operations. |

## Endpoint Classes

| Class | Description |
|-------|-------------|
| [Info](info.md) | Methods for retrieving market data, account information, and other read-only operations. |
| [Exchange](exchange.md) | Methods for placing orders, managing positions, and other trading operations. |
| [Subscriptions](subscriptions.md) | WebSocket subscription methods for real-time market data and account updates. |
| [Ws](ws.md) | WebSocket client for managing connections and accessing real-time interfaces. |

## Utility Classes

| Class | Description |
|-------|-------------|
| [Universe](universe.md) | Asset information, metadata, and utility functions for price/size formatting. |
| [Transport](transport.md) | Low-level transport classes for HTTP and WebSocket communication. |
| [Cloid](cloid.md) | Client order ID generation and management utilities. |

| Reference | Description |
